# TFG Theme — Image Assets

Drop the following images here (referenced by the templates). Until real
imagery is supplied, the templates will render broken-image placeholders,
which is expected in the staging phase.

Production images should be served from the WordPress media library
(`wp-content/uploads/`) via the Customizer or ACF; these local files are
fallbacks for the theme's static sections.
